# Note Taking Application
- This is a simple note taking application
- **FRONTEND**: React, Redux
- **BACKEND**: NodeJS, Express
- **Database**: MongoDB(NoSQL)

# to run locally:
- git clone -repo-
- npm install
- pm2 start process.config.js
- visit your localhost at port 3004

# requirements
- mongodb
- pm2
